# Express TypeScript template

# Pre-reqs

- Install [Node.js](https://nodejs.org/en/)
- Install Yarn

```bash
  npm install -g yarn
```
- Install Serverless

```bash
  npm install -g sls
```

# Getting started


- Install dependencies

```bash
yarn install
```

- Build and run the project offline

```bash
yarn build
yarn dev
```

Navigate to `http://localhost:4000/dev/status`
